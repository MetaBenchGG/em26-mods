# Roadmap

## Validated baseline

- BepInExPack EM2026 0.1.0.
- EM Framework 0.2.2.
- Better Game Data 1.1.0 in Shadow mode.
- AutoTraining 1.2.0 installed and verified in the current career.

## Known data boundary

Native match output contains map, teams, winner, score and player totals for kills,
deaths, assists, damage, headshots, KAST count, rounds, result and multikills.

Better Game Data Shadow mode generates independent synthetic round events. Those
events are useful for experiments but are not the hidden history of the native match.

## Development order

1. Ratings100: 0–100 player ratings and attributes as a presentation-only base mod.
2. AutoTraining redesign and staff-dependent automation rules.
3. Dynamic development and potential.
4. Dynamic market value, reputation and form.
5. Transparent HLTV-like match rating based on native totals.
6. Optional replacement match simulator with authoritative round events.
7. Persistent career database/wiki for rosters, transfers, matches, tournaments,
   rankings and records.
